## Design System Components

### Color Palette (Focus Mode)
```
Light Mode:
- Background: #FAFAFA
- Surface: #FFFFFF
- Primary: #2196F3 (Focus Blue)
- Success: #4CAF50
- Warning: #FF9800
- Text: #212121
- Muted: #757575

Dark Mode (Focus):
- Background: #0A0A0A (True black for OLED)
- Surface: #1A1A1A
- Primary: #64B5F6
- Success: #81C784
- Warning: #FFB74D
- Text: #FFFFFF
- Muted: #9E9E9E
```

### Typography
```
Headers: Inter or SF Pro Display
Body: Inter or SF Pro Text
Monospace: JetBrains Mono (for timers)
```

### Animations
- Smooth transitions (200-300ms)
- Subtle breathing animation for focus timer
- Gentle pulse for notifications
- Slide-in for distraction blocks
- Confetti for achievements

### Key Interactions
1. **One-Click Start**: Big, prominent CTA buttons
2. **Drag to Adjust**: Slider for time/energy
3. **Swipe Actions**: Mobile gestures for quick actions
4. **Keyboard Shortcuts**: Cmd+F for quick focus
5. **Voice Commands**: "Start deep work session"

---

## Summary

These mockups showcase a clean, focused interface that:
- **Reduces friction** for starting focus sessions
- **Provides clear feedback** on progress and achievements
- **Minimizes distractions** during active focus
- **Celebrates success** with meaningful analytics
- **Adapts to user patterns** with AI suggestions
